# Welcome to My Api
***

## Task
We need to build own api with some our interests theme data over 1K

## Description
The app works on 4K nba players data

Features:
- User authentification,session in Redis
- Get/Post/Update/Destroy, pagination 20 
- Cache (redis)

## Installation

```
npm install
```

